# Wilma Portfolio

Frontend developer portfolio built with React, TypeScript and Emotion.

This project focuses on creating clear, user-friendly interfaces with strong attention to structure, accessibility and maintainability.

## Tech

* React
* TypeScript
* Vite
* Emotion (CSS-in-JS)

## Focus

* UX and accessibility (WCAG)
* Clean and predictable user flows
* Scalable and maintainable component structure

## Status

Work in progress. Continuously being improved and expanded.
